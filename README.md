## Introduction
### 🐈 GilCat 길캣

길고양이 관리를 효과적으로 하고 싶은 캣맘과 캣대디에게 
안전하고 공유 가능한 개인 기록장을 제공하는 App 입니다.

## Functions
- 핵심적인 내용의 기록장
- 위치 노출 없는 길고양이 관리 공유
- 포근하고 정서적인 UI
